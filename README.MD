# Portfolio 
This portfolio is where contains my personal projects accessible on the internet.

## Built with the Following
- Vite v8.0.12
- React
- Javascript
- Javascript XML
- CSS3
- NPM

The project is fully organized with autoexplained folder names on `./src/`, and also includes a configurated deployment file for Github Actions.

```
./src/components
./src/js/
./src/json/
./src/pages/
```

To start to running locally, execute the code bellow. Is set to run on `http://localhost:3000/Portfolio/`
```
npm run dev
```
